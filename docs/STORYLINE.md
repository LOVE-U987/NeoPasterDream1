# PasterDream 模组完整剧情说明

## 概述

PasterDream 是一套以**梦境层级**为核心的剧情系统，玩家将从主世界出发，逐步深入染梦维度、灯影世界，最终抵达暗影地牢，揭开"无名"的身世之谜，并做出最终的抉择。

---

## 剧情世界观

### 核心设定

- **染梦维度（DyeDream World）**：第0层梦境，一切故事的开始。安全、宁静，是寻梦者们踏足的第一个异世界。
- **灯影世界（Lamp Shadow World）**：更深层的梦境，黑暗与光明交织。这里是无名守护的地方，也是暗影生物的巢穴。
- **暗影地牢（Shadow Dungeon）**：灯影世界深处的随机生成地牢，共5层，最底层藏着无名的秘密。
- **寻梦者**：曾经探索这些维度的前人，留下了笔记和遗迹。
- **San值**：精神状态指标，深入梦境会消耗San值，过低会导致精神失常。

### 核心主题

1. **梦境层级** — 从现实逐步深入潜意识深处
2. **记忆与存在** — 无名失去了自我，靠记忆维持存在
3. **诅咒与承诺** — 守护已故之人的坟墓
4. **暗影侵蚀** — 影子从梦境入侵现实
5. **选择与牺牲** — 最终的道德抉择

## 完整剧情流程

### 第一阶段：梦的开始（主世界）

#### 触发条件
- 玩家首次进入世界
- 模组自动授予隐藏成就 `achievement_start`【梦的开始】

#### 剧情内容
玩家在探索主世界时，会偶然发现一种奇特的方块——**染梦裂隙（DyeDream Crack）**。这种裂隙散发着微弱的光芒，似乎连接着某个未知的世界。

#### 关键物品
| 物品 | 说明 |
|------|------|
| `dyedream_crack` | 染梦裂隙方块，主世界罕见生成 |
| `dreamnotes_1` | 寻梦者笔记#1，记录初次梦境 |

#### 成就链
```
achievement_start（梦的开始）→ achievement_a_0（染梦裂隙）
```

---

### 第二阶段：染梦裂隙（主世界 → 染梦维度）

#### 触发条件
- 获得 `brokennotes_0`（破碎笔记0）
- 解锁配方 `crafting_215`

#### 剧情内容
玩家在睡梦中惊醒，背后冒出冷汗。回忆起梦中遇见的奇怪裂隙，在慢慢靠近并凝视着你。你拿起附近的材料迅速把这些梦境记录下来，在这之后你就失去了对这段梦境的记忆。

**"我必须知道发生了什么..."**

#### 进入染梦维度
| 条件 | 说明 |
|------|------|
| **物品** | 制作 `dreamjuice`（梦境果汁）|
| **效果** | 获得 `DREAMWISH_BUFF` 状态效果 |
| **触发** | 在夜晚睡觉 |
| **传送点** | (0.5, 108, 0.5) 或 (0.5, 158, 0.5) |

#### 成就
- `achievement_a_0`【染梦裂隙】— "获得一张灯影之下梦境的笔记残片"
- `achievement_c_1`【梦境果汁】— "制作并饮用梦境果汁，在睡梦中前往染梦之境"

### 第三阶段：染梦之境（染梦维度）

#### 维度特性

| 属性 | 说明 |
|------|------|
| **维度ID** | `pasterdream:dyedream_world` |
| **层级** | 第0层梦境 |
| **安全性** | 安全，无怪物生成 |
| **生物群系** | 染梦平原、染梦蘑菇林、染梦雪原、染梦冻洋 |
| **地表方块** | 染梦草方块、染梦沙、染梦泥土 |
| **默认方块** | 方解石（Calcite）|

#### 生物群系详情

| 生物群系 | 特征 |
|---------|------|
| `biome_dyedream_0` | 染梦平原 — 温和气候，基础生态 |
| `biome_dyedream_1` | 染梦蘑菇林 — 高湿度，蘑菇生态 |
| `biome_dyedream_2` | 染梦雪原 — 寒冷气候，冰雪覆盖 |
| `biome_dyedream_3` | 染梦冻洋 — 极寒，粉雪地表 |

#### 剧情内容

> "你在帕斯特之梦中将踏入的第一个群系，也就是第0层梦境。这里不会生成任何的怪物，不会有陷阱和危及你生命的存在。这里是一切故事的开始，你将会在这个世界中发现许多新事物和前辈寻梦者的遗迹，阅读他们留下的故事。"

#### 关键探索点

1. **浸染教堂**
   - 半空中黑白相间的教堂结构
   - 内有寻梦者笔记，记录主世界特殊坐标
   - 成就 `hide_10`：阅读寻梦者笔记"侵染教堂"

2. **融梦水晶箱**
   - 散布在世界各地的宝箱
   - 内含融梦水晶碎片（后续重要材料）
   - 成就 `achievement_treasure_dyedream`：染梦宝藏

3. **世界树**
   - 染梦维度的标志性巨型树木
   - 掉落世界树种荚

#### 关键资源

| 资源 | 用途 |
|------|------|
| `dyedream_dust` | 染梦粉尘，制作材料 |
| `dyedream_ingot` | 染梦合金锭，装备材料 |
| `dyedream_quartz` | 染梦石英，建筑材料 |
| `dyedream_fruit` | 染梦果，食物 |
| `meltdream_crystal` | 融梦水晶，高级材料 |

#### 成就链
```
achievement_b_0（访问染梦之境）→ achievement_c_0（获得染梦粉尘）→ achievement_c_1（饮用梦境果汁）
```

### 第四阶段：于影共眠（染梦维度 → 灯影世界）

#### 触发条件
- 完成 `achievement_shadow_start`（进入灯影之下）
- 制作暮影长床（Shadow Bed）
- 在夜晚或雷暴中使用

#### 关键物品：暮影长床

| 属性 | 说明 |
|------|------|
| **物品ID** | `shadow_bed` |
| **制作** | 需要染梦维度资源 |
| **使用条件** | 夜晚或雷暴 |
| **消耗** | 10点San值 |
| **传送目标** | `pasterdream:lamp_shadow_world` |

#### 剧情内容
玩家在染梦维度收集足够资源后，可以制作**暮影长床**。这是一张特殊的床，能够带领使用者进入更深层的梦境——灯影世界。

#### 成就
- `achievement_shadow_start`【灯影之下】— "深入世界中最黑暗的一面"
- `achievement_shadow_a_1`【于影共眠】— "利用自制的暮影长床前往灯影"

### 第五阶段：灯影世界

#### 维度特性

| 属性 | 说明 |

### 第六阶段：困顿囚徒（灯影世界 → 暗影地牢）

#### 触发条件
- 完成 `achievement_shadow_b_0`（获得黑金属）
- 找到暗影地牢传送门遗迹
- 结构完整（需要 `SHADOW_DUNGEON_BLOCK_0` + `SHADOW_DUNGEON_BLOCK_1`）

#### 传送门机制

| 属性 | 说明 |
|------|------|
| **结构要求** | 特定方块组合 |
| **冷却时间** | 1800秒（30分钟）|
| **传送倒计时** | 3秒倒计时 |
| **目标深度** | y-19 |

#### 地牢生成机制

暗影地牢共**5层**，每层随机选择结构模板：

| 层级 | 深度 | 随机数 | 结构模板 |
|------|------|--------|---------|
| Layer 1 | y-28 | 1-6 | `shadow_dungeon_0` ~ `shadow_dungeon_5` |
| Layer 2 | y-37 | 1-4 | `shadow_dungeon_10` ~ `shadow_dungeon_13` |
| Layer 3 | y-46 | 1-3 | `shadow_dungeon_7` ~ `shadow_dungeon_9` |
| Layer 4 | y-55 | 1-2 | `shadow_dungeon_6`, `shadow_dungeon_14` |
| Layer 5 | y-64 | 1-2 | `shadow_dungeon_npc_0`, `shadow_dungeon_npc_1` |

#### 成就
- `achievement_shadow_c_0`【困顿囚徒】— "进入暗影地牢"

## San值系统

### 机制
- 深入梦境会消耗San值
- 使用暮影长床消耗10点San值
- 使用真·暮影长床消耗10点San值
- San值过低会导致精神失常

### 相关效果
| 效果 | 说明 |
|------|------|
| `REST_BUFF` | 休息效果，睡觉获得 |
| `DREAMWISH_BUFF` | 梦境祈愿，进入染梦维度所需 |
| `SHADOW_SPYON_BUFF` | 暗影窥视，暗影入侵相关 |

## 技术实现要点

### 成就系统
- 所有剧情进度通过 Minecraft 原版的 `Advancement` 系统管理
- 使用 `minecraft:impossible` 触发器，由代码手动授予

### 维度传送
- 使用 `ServerPlayer.teleportTo()` 方法
- 配合 `ClientboundGameEventPacket` 等网络包实现平滑过渡

### 对话系统
- 使用 `queueServerWork(ticks, runnable)` 实现定时消息
- 通过 `Player.displayClientMessage()` 发送对话

### 地牢生成
- 使用 `StructureTemplate.placeInWorld()` 生成结构
- 每层随机选择模板，实现roguelike体验

## 总结

PasterDream 的剧情系统是一个精心设计的**多层梦境叙事**，玩家需要：

1. **探索** — 在主世界和染梦维度收集线索
2. **制作** — 合成关键物品（梦境果汁、暮影长床）
3. **深入** — 逐步进入更深层的梦境
4. **对话** — 与无名NPC交流，了解故事
5. **抉择** — 在最终做出灯与影的选择

整个剧情融合了**克苏鲁式恐怖**、**哲学思考**和**情感叙事**，为玩家带来独特的游戏体验。
