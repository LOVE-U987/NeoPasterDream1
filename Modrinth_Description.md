## 这，是什么？

**NeoPasterDream** 是经典 **PasterDream** 模组在 Minecraft **1.21.1 NeoForge** 上的**精神续作**。

原模组（1.20.1 Forge）作者创作的精美纹理、模型与视觉设计，我们完整保留——但**所有代码从零重构**，不使用任何 MCreator 生成代码。

> 🎨 **致谢** — 本模组所有纹理、模型与视觉设计均源自 **原 PasterDream 模组作者** 的创作。我们站在巨人的肩膀上。

***

## 等等，那不就是一个移植版吗？

准确地说——是**重生**。

原模组是用 MCreator 工具生成的，代码结构、性能和扩展性都受限。NeoPasterDream 用 **NeoForge 原生 API + DeferredRegister + GeckoLib** 重新实现了一切，让模组拥有更好的性能和更大的扩展空间。

但关键在于：**这东西还在长**。目前进度大概是……

## ❓ 你可能想问

**Q: 这跟原版 PasterDream 有什么区别？**
A: 原版是 1.20.1 Forge + MCreator 生成。NeoPasterDream 在 1.21.1 NeoForge 下完全重写，使用原生 API 实现。内容上覆盖了原模组大部分功能，未来还会有独占新内容。

**Q: 会不会换掉原模组的美术风格？**
A: 不会。原作者的纹理、模型、视觉设计是我们珍视的遗产，全部保留。

**Q: 进度条看着还没过半，现在能玩吗？**
A: 能。目前染梦维度已经可以探索，大部分生物可以遭遇，武器和工具可以使用。新冒险随时可以开始。

**Q: 存档会出问题吗？**
A: 染梦维度通过裂隙进入，不影响主世界。后续更新会保持向后兼容。

**Q: 能联机吗？**
A: 完全兼容多人服务器。
